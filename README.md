# musiccity

A complete Music Player Build in Flutter


The App inclues :-

            Fetching all the music Files from the local Storage
            Play/Pause/Seek
            Adding to Favourites
            Includes Local database (Hive database)
            Cool UI 
            Minimalistic Design
            Sleep Timer

Planning More Updates
